# 🚌 Pathak Brothers Bus Reservation System

A modern and responsive Bus Reservation System built with React and Firebase. The project allows users to search buses, book seats, manage bookings, download tickets as PDF, and provides a complete Admin Dashboard for route and booking management.

---

## 🚀 Live Demo

https://pathak-brothers-bus-reservation.netlify.app

---

## ✨ Features

### 👤 User Features

- 🚌 Search Available Buses
- 💺 Live Seat Selection
- 👥 Multiple Passenger Booking
- 🎫 Booking Confirmation
- 📄 Download Ticket as PDF
- 🖨️ Print Ticket
- 📱 QR Code on Ticket
- 📖 My Bookings
- ❌ Cancel Booking
- 📱 Fully Responsive Design

---

### 🔐 Admin Features

- Admin Login
- Dashboard Overview
- Booking Statistics
- Revenue Statistics
- Route Management
- Add Route
- Edit Route
- Delete Route
- Booking Search

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- Firebase Firestore
- React Router DOM
- jsPDF
- QRCode React

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/rajchoubey19/pathak-brothers-bus-reservation.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
