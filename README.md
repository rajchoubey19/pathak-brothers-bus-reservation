# 🚌 Pathak Brothers Bus Reservation System

A modern and responsive Bus Reservation System built using React, Node.js, Express.js, and MongoDB. The project allows users to search buses, select seats, book tickets, and download a printable PDF ticket.

## 🌐 Live Demo

https://pathak-brothers-bus-reservation.netlify.app/

## ✨ Features

- 🔍 Search available buses
- 🚌 Route selection
- 💺 Interactive 2+3 seat layout
- 👤 Passenger details form
- 🎫 Booking confirmation
- 📄 Download ticket as PDF
- 🖨️ Print ticket
- 📱 Responsive design
- 🎨 Modern UI

## 🛠️ Tech Stack

- React
- JavaScript
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
